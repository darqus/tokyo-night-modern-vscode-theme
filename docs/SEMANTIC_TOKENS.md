# Модуль семантических токенов

**Дата создания:** 2025-11-03

Модуль семантических токенов предоставляет расширенную поддержку для различных языков программирования и фреймворков, позволяя точно настраивать синтаксическую подсветку в соответствии с семантическим значением элементов кода.

## Структура модуля

Модуль семантических токенов организован в следующие файлы:

- [`functions.ts`](src/theme/generator/semantic/functions.ts:1) - определяет токены для функций, методов их компонентов
- [`variables.ts`](src/theme/generator/semantic/variables.ts:1) - определяет токены для переменных, констант и параметров
- [`modern-frameworks.ts`](src/theme/generator/semantic/modern-frameworks.ts:1) - определяет токены для современных фреймворков и библиотек

## Поддерживаемые технологии

Модуль предоставляет поддержку для следующих языков и технологий:

- **React** - компоненты, хуки, JSX-элементы
- **Vue** - компоненты, директивы, шаблоны
- **Rust** - функции, типы, трейты, макросы
- **Go** - функции, методы, интерфейсы, структуры
- **Python** - функции, классы, переменные, аннотации типов
- **GraphQL** - запросы, мутации, типы, поля
- **TypeScript/JavaScript** - классы, интерфейсы, типы, модули
- **CSS/Sass/Less** - селекторы, свойства, переменные
- **JSON/YAML** - ключи, значения, структуры данных

## API для создания семантических токенов

Для создания семантических токенов используется функция `createSemanticToken`:

```typescript
function createSemanticToken(
  tokenType: string,
  tokenModifiers?: string[],
  color?: string
): SemanticToken
```

Параметры:

- `tokenType` - тип токена (например, "function", "variable", "class")
- `tokenModifiers` - модификаторы токена (например, ["declaration"], ["readonly"], ["static"])
- `color` - цвет, который будет применен к токену

## Примеры использования

```typescript
// Создание токена для функции
const functionToken = createSemanticToken("function", ["declaration"], "#70a5fd");

// Создание токена для переменной
const variableToken = createSemanticToken("variable", ["readonly"], "#abb2bf");

// Создание токена для класса
const classToken = createSemanticToken("class", ["declaration", "export"], "#e5c07b");
```

Эти токены затем используются в генераторе темы для определения цветов элементов кода в зависимости от их семантического значения, обеспечивая более точную и выразительную подсветку синтаксиса.
